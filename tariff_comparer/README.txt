Телеграм-бот для Сберты с функционалом сравнения тарифов.
1) Для работы необходим Python версии 3.7, и pip версии >18.0. Правильная работа на других версиях не гарантируется.
2) При первом запуске: запустить install_requirements_windows.bat (для Windows)/install_requirements_xnix.sh (для unix-like систем).
3) Для запуска бота на территории РФ скорее всего понадобится VPN, установите любую программу/настройте VPN самостоятельно.
4) Запустите бота скриптом run_windows.bat (для Windows)/run_xnix.sh (для unix-like систем).
